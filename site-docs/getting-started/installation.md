# Installation

## Requirements

- **Python 3.10+** - [Download Python](https://python.org)
- **Docker** - Required for database and services
- **Node.js 18+** - For frontend development

## Install Lich CLI

### From PyPI (Recommended)

```bash
pip install lich
```

### Verify Installation

```bash
lich --version
# ðŸ§™ Lich Framework v1.4.1
```

### Update to Latest Version

```bash
pip install --upgrade lich
```

## Optional Tools

For the best development experience, install these tools:

### IPython (for `lich shell`)

```bash
pip install ipython
```

### pytest-watch (for `lich test --watch`)

```bash
pip install pytest-watch
```

## Next Steps

After installation, create your first project:

```bash
lich init
```
